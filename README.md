# 💫 About Me

👨🏻‍💻 I’m a full-stack developer with a strong foundation in **.NET, DevOps, and cloud computing**.

👨🏻‍🎓 I’m currently studying **Computer Engineering at Middle East Technical University**.

📱 I’m working on **Deviny**, a fitness-focused startup I’m building with a team as a co-founder. This is a social platform that connects users with trainers and nutritionists. The platform includes scheduling, messaging, fitness program purchases, and tools for managing training and nutrition services.

🎧 I’m also building a multimodal AI project that analyzes books and narration to create context-aware listening experiences. It identifies elements such as mood, pacing, intensity, and scene context, then matches them with suitable music and sound effects from an existing audio library. The system also handles transitions between tracks to maintain a coherent atmosphere.

🔭 I’m currently exploring **Semantic Kernel, Hugging Face Transformers, and LangChain**, with a focus on multimodal AI systems and intelligent application development.

# 🚀 Featured Projects

* **[MedAnnotateApp](https://github.com/ArkhunAbdullazade/MedAnnotateApp)** — A medical image annotation platform developed during AI-for-medicine research involving Stanford University and the Hasso Plattner Institute. It provides separate role-based workflows for medical students and professionals, with tailored annotation interfaces, clinical-term highlighting, canvas-based drawing tools, filtered image assignment, timing capture, and image locking for controlled research studies. Built with **.NET 8, ASP.NET Core MVC, Entity Framework Core, PostgreSQL, Docker, and Redis**.

* **[MangaFlex](https://github.com/ArkhunAbdullazade/MangaFlex)** — A team-built manga discovery and reading platform powered by the MangaDex API. It supports title search, detailed manga information, in-browser chapter reading, user authentication, and recently read history. Built with **ASP.NET Core MVC, Entity Framework Core, PostgreSQL, ASP.NET Core Identity, MediatR, and xUnit**.

* **[Deviny](https://github.com/Deviny-me/Deviny_IOS)** — A fitness-focused social platform. Deviny connects users with trainers and nutritionists through social features, messaging, appointment scheduling, fitness-program purchases, and tools for managing training and nutrition services. The linked repository contains the **Swift iOS application**, while the platform’s backend services and other components are maintained within the Deviny organization.

* **[Spotibuds](https://github.com/Spotibuds/User)** — A team-built social music platform developed using a microservice architecture. My primary contributions covered user profiles, social relationships, real-time chat, notifications, feed activity, and Azure Blob Storage integration for user media. I also collaborated on the **Music and Identity services, frontend development, and Docker containerization**. The platform uses **.NET 8, ASP.NET Core, MongoDB, SignalR, RabbitMQ, Azure services, and JWT authentication**.
